Help

Edit an tile ( App or Favorite )

click with the right mouse button on the tile you like to edit.
At the bottom of the new tab you see now an blue edit panel, there you can modify the link, or remove the tile or app from chrome.

Add new tile/favorite

click on the top right of the screen on the settings icon.
Now it opens an new Settings Menue, choose there add new tile/favorite and you be able to add an new tile to the new tab page.
Tip:
You can drop an shortcut from chrome into the "add new favorite/tile" formular and the link will be automatically added
